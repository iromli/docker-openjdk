# Overview

Eclipse Temurin (OpenJDK) on Alpine with `glibc` support.

## Versions

Supported variants:

- `jre`
- `jdk`

Supported OpenJDK versions:

- OpenJDK 11.0.14

## Building Image

Go to the corresponding directory (i.e. `11`) and run `make build-jre` to build JRE variant or `make build-jdk` for JDK variant.
